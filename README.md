# Malki Amasha — Portfolio

A personal portfolio website built with React and Vite, showcasing my work as a Data Analyst.

## Live Sections

- **Hero** — Introduction with animated avatar and tagline
- **About Me** — Bio, skills (hard & soft), tools, and curved intro text
- **Work Experience** — Professional timeline
- **Skills** — Technology and tool proficiency
- **Featured Projects** — Highlighted project showcases
- **Contact** — Get in touch section with social links

## Tech Stack

- **React 19** — UI framework
- **Vite 8** — Build tool with HMR
- **CSS** — Custom styling with animations (glowing particles, twinkling background)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
portfolio/
├── public/images/     # Static assets (avatar, QR code, skills, etc.)
├── src/
│   ├── components/    # React components (Hero, About, Contact, etc.)
│   ├── App.jsx        # Main app layout
│   ├── App.css        # Global styles & background animations
│   └── main.jsx       # Entry point
├── index.html
└── vite.config.js
```
